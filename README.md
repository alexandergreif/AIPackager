# 🤖 AIPackager

**AI-powered script generation for automated software packaging using the PowerShell App Deployment Toolkit (PSADT).**

---

## 💡 Project Overview

**AIPackager** is a tool designed to simplify and accelerate 🏎️ software deployment workflows by generating ready-to-use PSADT scripts with the help of AI 🤖. 

Targeted at IT teams and system administrators 🧑‍💻, the tool aims to:
- Reduce manual effort ✂️
- Ensure consistency ✅
- Streamline enterprise application packaging 📦

Future-ready as a potential SaaS or on-premise solution, **AIPackager** is tailored with European compliance standards in mind 🇪🇺.

---

## 🚀 MVP Features (Phase 1)

- 📤 Upload MSI or EXE installers via a simple web interface.
- ⚙️ Define key deployment parameters (e.g., silent install flags, target paths).
- 🧠 Generate PSADT scripts using AI (OpenAI API).
- 📝 Export scripts and basic documentation to Git version control.

---

## 🔮 Coming Soon (Phase 2)

- 🧠 Context-aware AI using internal docs and best practices via LangChain.
- 🔍 Semantic search and retrieval-augmented generation (RAG) with ChromaDB.
- 🏢 Enterprise-specific, compliance-aware script suggestions.
